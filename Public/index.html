About


The PokeRaidTeam app was created by CatR3kd for fun, but turned into a fully-functional app for finding raid teams in Pokemon Go.


To use this website, you can host or join players in raids in Pokemon Go by clicking on any of the current raids and joining via the provided friend code or hosting a raid yourself.
Some things to note:


  • Do not send strangers in Pokemon Go gifts, as they will be able to see where you are sending them from.
  • If you host a raid, all players will be able to see where you are hosting it from.
  • Our website uses third-party API's to obtain raid information, and some or all information may be updated late.
